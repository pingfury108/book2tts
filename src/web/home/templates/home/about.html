关于 Book2TTS

专为图书制作人打造的在线工作台，从电子书解析到音频发布一站式完成

书籍工作台

提供目录/页面双视图、原文预览与资源提取，精细掌控每一页内容。

LLM 智能处理

集成排版、翻译、对话脚本、章节提取等工具，自动记录 token 与缓存命中。

音频与发布

支持多语音合成、字幕与章节生成，生成标准 RSS 订阅方便播客分发。

章节 & RSS

章节自动生成并支持手动微调，RSS feed 随音频更新，兼容主流播客客户端。

如何使用

1

上传书籍

支持 EPUB、PDF 等格式，工作台自动解析目录和页面。

2

挑选与润色

在工作台中勾选章节，按需调用排版、翻译或对话生成工具。

3

生成并发布

启动音频、字幕、章节任务，完成后可下载或发布到 RSS。

功能亮点

🔊 工作台全景

一页掌握章节、页面、音频状态与原文件资源。

  • 目录 / 页面双列表，支持多页批量操作
  • 原文、图片、资源一键预览与下载
  • 音频状态面板显示进度、失败原因与建议

🎯 LLM 助手

排版、翻译、对话脚本、章节提取随取随用。

  • 多模型接入，缓存命中免扣积分
  • 对话脚本分片处理，可断点续跑
  • 自动记录 token、模型与失败信息

🌐 分发与收听

生成的音频可在线预览、下载或通过 RSS 发布。

  • 支持多音色合成、字幕和章节文件导出
  • 按用户、书籍或 Token 生成 Podcast 兼容 RSS
  • 嵌入式播放器，便于团队内部审核

📡 章节与 RSS

章节自动生成，整合到 RSS feed 中方便播客客户端识别。

  • 支持字幕驱动的章节提取，可手动调整
  • 章节信息随音频一起写入 RSS，兼容 Apple Podcasts 等平台
  • 章节与 RSS 资源可下载归档，方便二次分发

常见问题

📁 支持哪些文件格式？

工作台当前支持 EPUB 与 PDF，PDF 会自动区分文本版和扫描版并提供 OCR 工具。

🗂️ 章节生成需要哪些条件？

系统优先使用字幕（SRT）生成章节；若无字幕，可改用 LLM 解析，但准确度依赖文本质量。

📡 RSS 支持哪些客户端？

生成的 feed 兼容 Apple Podcasts、Pocket Casts 等主流客户端，章节信息会一并展示。
